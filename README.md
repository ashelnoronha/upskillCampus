## IoT-Based Precision Irrigation Management System Using ESP32

## Overview
This project implements a smart irrigation system using an ESP32 microcontroller. It automates irrigation by monitoring soil moisture, rainfall, ambient light, temperature, and humidity to optimize water usage.

## Features
- Automatic irrigation control
- Soil moisture monitoring
- Rain detection
- Temperature-based irrigation
- Humidity-based irrigation
- Day/Night detection using LDR
- OLED display with multiple information screens
- Buzzer notification when irrigation starts

## Hardware Components
- ESP32
- DHT22 Temperature & Humidity Sensor
- SSD1306 OLED Display
- LDR
- Soil Moisture Sensor (Potentiometer in Wokwi)
- Rain Sensor (Slide Switch)
- Buzzer
- LED (Pump Indicator)

- Arduino IDE
